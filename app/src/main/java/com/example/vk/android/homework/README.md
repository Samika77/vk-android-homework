# RuStore App Catalog

Android‑приложение в стиле RuStore: показывает список приложений и детальную информацию о конкретном приложении при клике на него.

## Технологии

- Kotlin
- Jetpack Compose
- Navigation Component (NavGraph)
- Clean Architecture, MVVM
- Coroutines
- Hilt (DI)
- Retrofit (REST‑API)
- Room (локальное кэширование)

## Функционал

- Список приложений с базовой информацией.
- Экран деталей приложения при клике на элемент.
- Кнопка «Поделиться» — вызывает стандартный шаринг системы.
- Кнопка «Установить» — заглушка события, которая в будущем должна открыть установку через Package Installer.
- Обработчики кликов реализованы через Channel: события из UI уходят в ViewModel.
- UI‑эффекты (показ Snackbar) реализованы через `LaunchedEffect`, который отслеживает события из ViewModel.

## Как запустить

1. Открыть проект в Android Studio.
2. Дождаться загрузки Gradle и зависимостей.
3. Выполнить сборку (Build → Rebuild Project).
4. Запустить приложение на устройстве или эмуляторе.

## Архитектура

- UI: Jetpack Compose.
- Навигация: два экрана через NavGraph (список и детали).
- ViewModel: управляет состоянием экрана и событиями.
- Events: клики и другие действия пользователя передаются через Channel.
- Data: Retrofit для сети, Room для кэша.
- DI: Hilt на всех уровнях.
